# Тестовое задание для Mafin

React приложение, использующее открытое API Rick & Morty, 
отображающее всех персонажей с возможностью фильтрации по имени,
а также их кратким описанием при наведении.

API: https://rickandmortyapi.com/documentation/ 

Установка и запуск:
```
git clone https://github.com/ddnwh/mafin-test-task.git
npm i
npm start
```
